Stop postponing. Start tracking. Keep planning.

All your favourite time managing and organizational tools in one app.

User persona

User stories

  • As a user, I want to keep an agenda of all my meetings and plans so that I don't miss the important ones
  • As a user, I want to make grocery and todo lists so that I don't forget anything
  • As a user, I want to add a list of my most frequent habits so that I can easily remember to do them all
  • As a user, I want to organise my expenses so that I can save money for things that make me happy
  • As a user, I want to write down thoughts so that I can reread them later
  • As a user, I want to get enough sleep and be well rested so that I can achieve all my goals during the day
  • As a user, I want to drink enough water so that I can stay healthy all the time
  • As a free user, I want to use the limited package of the app so that I can see if it changes something in my daily activities
  • As a premium user, I want to have more tracking features so that I can use the app at its full potential
  • As a custom user, I want to personalize my tracking habits so that I can only follow what I am interested in

Use cases

Interviews conclusions

All people that we interviewed wished for a simple app, one that is easy to interact and to give quick access to functionalities. Most of them wanted an app to incorporate variated tools so that they don’t need to install multiple tracking apps. Mainly they wanted it to include to do lists, habit tracking tools, and also tools for budget and time management improvement.

This is why we opted for a simple interface with all the tools displayed on the main page. When opening the app, the user is shown an overview of all the sections each with a brief description of the content. From here we give easily access to a todo list, a planning calendar, a habit tracker, a budget management tool, sleep and water trackers and a tool for adding other important notes.

Clicking on each of the sections the user is redirected to that specific content. There he has variate options to add, edit and delete, mark or rate. Details are displayed for a specific date in a very simple way, integrating interactive diagrams and customizing options for a better management.
